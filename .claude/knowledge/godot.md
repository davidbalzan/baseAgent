# Godot Learnings

Accumulated knowledge about Godot game engine from project experience.

## Entries

- [2026-01-26] This project uses Godot 4.5+ which has significant differences from Godot 3.x - use class_name for type declarations, @export/@onready annotations, typed arrays (Array[Type]), new shader syntax (render_mode instead of hints), Decal3D instead of decal projectors, GPUParticles3D, and newer node types like SubViewport
